## Run Locally

Clone the project

```bash
  git clone https://github.com/Reactive-Karan/safle-dapp.git
```

Go to the project directory

```bash
  cd safle-dapp
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```

## Running Tests

To run tests, run the following command

```bash
  npm run test
```
